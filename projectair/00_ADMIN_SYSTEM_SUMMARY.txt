🔒 SECURITY & ACCESS CONTROL
───────────────────────────────────────────────────────────────

  ✅ Admin-Only Routes Protected:
     • /admin                    (blocked for regular users)
     • /admin/bookings           (blocked for regular users)
     • /admin/users              (blocked for regular users)
     • /admin/flights            (blocked for regular users)

  ✅ Authentication Required:
     • Must be logged in to access any route
     • Must have is_admin=1 to access admin routes
     • Regular users get error message if they try

  ✅ Session-Based Access:
     • Admin access tied to user session
     • Logout clears admin session

  ✅ Password Security:
     • All passwords hashed with werkzeug
     • No plain text passwords stored
     • Safe database queries (no SQL injection)

🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────

  Q: Admin Panel button not visible?
  A: • Make sure you're logged in as admin
     • Admin account must have is_admin=1
     • Try refreshing page

  Q: Can't access /admin route?
  A: • Only admins (is_admin=1) can access
     • Regular users get redirected
     • Logout and login as admin

  Q: Email already exists error?
  A: • Use different email for admin account
     • Or make existing user admin via database

  Q: Database migration failed?
  A: • Column already exists (this is OK)
     • Continue with creating admin account

  → See ADMIN_COMPLETE_GUIDE.md for more solutions
