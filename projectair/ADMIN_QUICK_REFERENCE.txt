                       🔒 SECURITY FEATURES

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ Admin-Only Routes Protected
     Regular users cannot access /admin routes

  ✅ Role-Based Access
     Only users with is_admin=1 get admin features

  ✅ Secure Passwords
     All passwords hashed with werkzeug

  ✅ Session-Based Auth
     Session required to access any admin feature

  ✅ Input Validation
     Database queries use safe parameters
